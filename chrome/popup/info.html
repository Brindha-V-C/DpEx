About

"DpEx", a browser extension developed to help consumers navigate the internet . The extension highlights the individual dark patterns on websites so that consumers become aware of the influences impact them. The tool provides information regarding the nature of these patterns and allows the user to block the dark patterns if necessary (User's choice).

What are Dark Paatterns?

E-commerce dark patterns refer to deceptive or unethical user interface design choices made by online retailers to manipulate or mislead users into taking actions that they might not otherwise intend to take. These practices are often employed to increase sales, capture user data, or achieve other business goals. Dark patterns can violate principles of transparency, user trust, and ethical business conduct.

Instructions

• Report – Use the report button to report a dark pattern

• Block – Use the block button to remove the patterns from the website. If you want to retrieve it back just reclick the button

• Power on/off – If you press the power button, the extension will be disabled and if want to enable it again just enable the extension in manage extension page of your browser.
